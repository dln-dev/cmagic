double staticFac ( unsigned short int  n)

this implementation keeps track of the last factorial calculated. Resets automatically, when used in a new loop (n = 1). ONLY USE IN LOOPS! It won't return 0!, so you have to take care that your loop starts such that 1! is the first factorial needed. See testfile for examples.

tpoint timestamp ( )

this function returns a std::chrono::high_resolution_clock::time_point marking the current time. Use this twice and calculate the difference with "duration(tpoint t1, tpoint t2)" to get the time a calculation took.
